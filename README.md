# Nucleo-F429ZI Board Support Package

[Board Information](https://os.mbed.com/platforms/ST-Nucleo-F429ZI/)

## Board Features
* 3 User LEDs
  * Green LED on PB0
  * Blue LED on PB7
  * Red LED on PB14
* User button on PC13
* USART3 to ST-Link Virtual Com Port
  * Tx: PD8
  * Rx: PD9
* 10/100 MBit Ethernet (RMII)
  * MAC: PA1/7, PB13, PC4/5, PG11/13
  * MIIM: PA2, PC1
